The Box Model

Properties
Width and Height Width and Height are the dimensions of an HTML box that will hold raw content inside the box
Padding Padding is like the space between a picture and its surrounding frame; it allows content to be expanded so it appears less cramped
Border Borders are like the frame around a painting, surrounding an element with a line
Margin Margin is used to expand the space outside of the box, creating space between different elements' content
